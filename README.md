# 🎬 Movie Recommender System

A content-based movie recommendation system built using **Python**, **Streamlit**, **Scikit-learn**, and the **OMDb API**. The application recommends movies similar to the one selected by the user and displays their posters through an interactive web interface.

---

## 🚀 Features

- Recommend the top 5 movies similar to the selected movie
- Interactive web interface built with Streamlit
- Displays movie posters using the OMDb API
- Content-based recommendation using cosine similarity
- Fast movie search through a dropdown menu

## ⚙️ How It Works

1. Load the movie dataset.
2. Preprocess movie features.
3. Convert movie information into numerical vectors.
4. Compute cosine similarity between movies.
5. Store the processed movie data.
6. When a user selects a movie, the application retrieves the most similar movies.
7. Fetch movie posters using the OMDb API.
8. Display the recommendations in the Streamlit application.

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/bhavanikamalekar/movie-recommender-system.git
```

Move into the project directory:

```bash
cd movie-recommender-system
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
